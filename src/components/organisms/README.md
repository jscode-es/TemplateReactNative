# Organisms

Los **organismos** son secciones de UI más grandes, formadas por moléculas y átomos.  
Tienen un propósito más definido dentro de la pantalla.

## Ejemplos

-   `Header` con logo, buscador y acciones.
-   `Navbar` con botones de navegación.
-   `Form` con múltiples inputs y botones.

## Buenas prácticas

-   Combinar átomos y moléculas de forma coherente.
-   No acoplarse a un caso de uso específico de negocio.
-   Testear interacciones principales (ej. envío de formulario).
